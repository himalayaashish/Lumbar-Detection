#### Lumbar Detection Using Machine Learning
###### This project focuses on detecting Lumbar bones using both instance segmentation and semantic segmentation techniques.
###### I utilized advanced machine learning techniques, applying the Mask R-CNN algorithm for instance segmentation and UNet for semantic segmentation. After evaluating the models, UNet achieved the best performance in terms of accuracy and segmentation quality for Lumbar bone detection.

###### To know the best model with best parameter I have used maskRCNN algorithmns and UNET has scored the best.
